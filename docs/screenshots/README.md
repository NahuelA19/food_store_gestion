## Instrucciones para screenshots

Para capturar el estado actual de la aplicación:

```bash
# Inicia el servidor de desarrollo
npm run dev

# Navega a cada pantalla en http://localhost:5173
# Toma screenshots usando tu herramienta favorita (Print Screen, PrtScn, etc.)
# Guarda cada screenshot con el nombre descriptivo (01-login.png, etc.)
```

## Notas

- Los screenshots deben mostrar la funcionalidad en **modo oscuro** (dark mode) para consistencia visual
- Asegúrate de que todos los elementos estén visibles y claros
- Los nombres de archivo deben coincidir con los listados en la tabla anterior
